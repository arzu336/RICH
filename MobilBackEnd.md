# Mobil Backend REST API Baglantisi Gorev Dagilimi

Bu dokumanda mobil uygulamanin REST API ile iletisimini saglayan backend entegrasyon gorevleri listelenmektedir. Her grup uyesi kendisine atanan REST API backend ve frontend baglantilarini kodlamak ve kanit videosunu kendi sayfasina eklemekle sorumludur.

## Genel Mobil Backend Prensipleri

- Mobil uygulamadan REST API'ye istek gittigi kanit videosunda net olarak gorunmelidir.
- Her gereksinim icin once gereksinim adi sesli olarak soylenmeli, sonra API baglantili islem gosterilmelidir.
- Basarili ve hatali durumlar kullaniciya anlasilir mesajlarla gosterilmelidir.
- Kayit, giris, cikis, hesap silme ve kategori urun listeleme islemleri REST API ile baglantili calismalidir.
