# RICH Web Frontend 🚀

Bu proje, **RICH** uygulamasının frontend (kullanıcı arayüzü) kısmıdır.  
React ve Vite kullanılarak geliştirilmiştir.

## 🛠️ Kullanılan Teknolojiler

- React
- Vite
- JavaScript / TypeScript (opsiyonel)
- CSS / SCSS

## ⚙️ Kurulum

Projeyi çalıştırmak için aşağıdaki adımları takip et:

```bash
npm install
npm run dev
